## About The Project

There were several challenges in these mini projects that we handled. 

![22](https://github.com/user-attachments/assets/f83967bf-ea8c-4703-895e-3b2a3fded9e7)

![21](https://github.com/user-attachments/assets/af0ae4ed-9cf7-4e7e-b39f-940ffef1120f)

This project taught me how to utilize CSS and HTML. 
React Dom, React Router Dom, Open API, Clear Authentication, Frontend and Backend Connection, MongoDB, Mongooes, Node.js, and Express.js

Here's why:
* HTML Multiple tags.
* Block versus inline elements
* CSS colors, CSS box properties.
* CSS variable classes, and pseudo classes
* Flex CSS and grid CSS annotation 
* CSS media query 
* The fundamental idea of JavaScript
* React js hooks like useState and UseEffect or many more...
* how to fetch data using axios
* how interact with Open API of Gemini
* ImageKit.io to store images
* Authentication using Cleark 

Naturally, not every topic that I discuss in the code above is mentioned. Thank you.
